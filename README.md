JEE Progress Tracker
A simple and interactive website to track daily preparation for JEE.
Features


Track daily study hours


Record Physics, Chemistry, and Maths problems solved


Add topics with completion deadlines


Analyze mock test scores


View progress using charts and graphs


Study streak calendar view


Responsive design for desktop and mobile


Data stored locally in browser using Local Storage



Technologies Used


HTML


CSS


JavaScript


Chart.js



How to Run


Download or clone this repository

index.html
in any browser
No installation required.
